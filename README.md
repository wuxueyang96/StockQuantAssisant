# StockQuantAssistant

A股实时股价查询工具

## 功能

- 支持通过公司名称查询A股实时股价
- 支持通过股票代码查询A股实时股价
- 自动判断是否为A股上市公司
- 多数据源支持（腾讯财经、新浪财经）

## 使用方法

```bash
# 通过公司名称查询
python get_a_stock_price.py 阳光电源

# 通过股票代码查询
python get_a_stock_price.py 300274

# 查询其他公司
python get_a_stock_price.py 贵州茅台
python get_a_stock_price.py 腾讯控股
```
